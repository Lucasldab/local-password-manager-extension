Step 1: Install Local Password Manager

First, make sure you have the local password manager CLI installed. Clone and build the repository:

git clone https://github.com/Lucasldab/local-password-manager.git cd local-password-manager cargo build --release

Make sure the binary is in your PATH or note its location.

Step 2: Start the Bridge Server

The bridge server allows the extension to communicate with your local password manager CLI. Start it using:

cd server ./start_server.sh --cli-path /path/to/local-password-manager

Or manually with Python:

python3 server/bridge.py --cli-path /path/to/local-password-manager

The server will start on localhost:8080 by default.

Step 3: Configure the Extension

Now configure your extension settings:

Path where your password vault will be stored

Step 4: Start Using the Extension

Once configured, you can:

  • Click the extension icon to manage passwords
  • Use Ctrl+Shift+L to auto-fill credentials on websites
  • Add new credentials through the extension interface
  • Search and retrieve stored passwords
